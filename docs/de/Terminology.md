# Terminology

* Domain name (name): A domain name that is easy to read and write, such as test.cat. Similar to the Internet's DNS, a domain name consists of multiple domains, which are separated by decimal points.
* Owner: The owner of the domain name.
* Registration: The process of adding domain-related information to the chain through smart contracts.
* Registrant : The owner of the domain name, who can transfer the domain name.
* Controller: The controller of the domain name, who can modify the resolver or reverse records.
* Resolver: A smart contract based on Ethereum, used to realize the mapping from domain name to address. The system provides a default parser, and users can also customize the parser.
